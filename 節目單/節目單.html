因著緣分，我們相遇；

因著相識，我們並肩同行；

因著珍惜，我們承諾彼此長久相挺；

因著工作相遇的我們，

在音樂裡找到共鳴。

在生活裡彼此分享，在困境中彼此扶持，在平凡的生活裡留下珍貴的回憶。 於是一人一支豎笛，在各式各樣的練習中，像千層蛋糕堆疊出一致的默契，像打火石碰撞出音樂的火花，在每個人臉上，綻放出最純粹的享受。 「愉樂」- 便是在這樣自然又真摯的流動中，誕生了。

相遇&成長

歷經了愉樂1、2和3後，很開心再次邀請你／妳，一同走進我們的重奏音樂會。 這一次，我們邀請到薩克斯風的夥伴們共襄盛舉，讓音樂會的色彩更加豐富，讓相遇的故事延續得更美。

感謝&感恩

有你／妳願意與我們共度這段美好夜晚，是對愉樂最大的支持。 未來旅程漫長，願我們心中都有自己喜愛的音樂，以此為光，成為路途上溫柔前行的力量。 今日，願與你／妳一起譜下的回憶，能靜待它的滋長，成為彼此的陪伴。

Por Una Cabeza

S.Sax. 吳政軒 / A.Sax. 黃翎茵
T.Sax. 余宛恩 / B.Sax. 許紘岳

是由阿根廷傳奇歌手Carlos Gardel作曲，Alfredo Le Pera 填詞的探戈歌曲，改編於Nick Keyes，原曲創作於1935年。 歌名直譯為「差一個頭（的距離）」，這是一個源自賽馬術語，意指馬匹在終點線前僅以「一個馬頭的距離」分出勝負。這首曲子是探戈的經典之一，旋律深情、充滿戲劇性。原曲搭配小提琴、鋼琴與班多鈕手風琴的演奏，展現濃厚的拉丁情懷與感性張力。 這首曲子表面上以賽馬為題，但實際上是在描寫一位男子在愛情與賭博間糾結的情感掙扎。歌詞將賽馬的「輸贏」隱喻為男人對女人的迷戀與沉淪，像賽馬場一樣讓人難以自拔。

Premier Quartet op.53

S.Sax. 吳政軒 / A.Sax. 黃翎茵
T.Sax. 余宛恩 / B.Sax. 許紘岳

由法國浪漫時期作曲家Singelée所作，並由薩克斯風大師Jean‑Marie Londeix重新編寫，這首為薩克斯風四重奏的代表作之一。作品以B♭大調為主，四個樂章洋溢著優雅而生動的氣息：

Ⅰ. Allegro moderato
歡欣而明朗的開端，兩支中音與高音樂器交錯推出主題，節奏活潑，充滿浪漫時期的典雅風采。

Ⅱ. Andante cantabile
柔和歌唱般的緩板，旋律溫暖流暢，如在對話的心聲裡，帶出四支樂器優美的合聲。

Ⅲ. Scherzo: Vivace
輕快俏皮的曲風，如精靈舞步般跳躍；其中短暫的Trio段落，又轉為柔和細膩，形成鮮明對比。

Ⅳ. Finale: Allegro con brio
熱情奔放的回旋終曲，短小動機層層推進，最後以華麗的快速段落劃下勝利句點。

整部作品結合了浪漫時期的抒情與早期新古典的刻板結構，既有華麗的技巧展示，也有柔美的合奏對話，邀請您一同感受19世紀中葉薩克斯風獨特的魅力與浪漫風情。

Chinese Rag

S.Sax. 吳政軒 / A.Sax. 黃翎茵
T.Sax. 余宛恩 / B.Sax. 許紘岳

Jean Matitia的《Chinese Rag》是一首為薩克斯風四重奏所作的輕快樂曲，融合了ragtime的節奏風格與東方音樂元素，展現出獨特的跨文化音樂語言。 這首以 ragtime 為基礎，是一種源自美國的音樂風格，以其切分節奏和活潑的旋律聞名，Matitia 在此基礎上，融入了東方音樂的元素，如五聲音階和模仿中國傳統。

In the Mood

S.Sax. 吳政軒 / A.Sax. 黃翎茵
T.Sax. 余宛恩 / B.Sax. 許紘岳
Gt. 王浩宇 / Per. 李品樺

是一首經典的 Swing 爵士樂曲，由 Joe Garland 作曲，它的旋律輕快活潑、節奏強烈，很容易讓人聯想到愉快、輕鬆甚至略帶調情的情緒。 最早由 Glenn Miller Orchestra 演奏因而走紅，可說是 Glenn Miller Orchestra 的成名之作，這也是最能代表40年代搖擺曲風的曲子。In The Mood 曾在1940年高居榜首長達13周之久，並且在被用在電影 Sun Valley Serenade 中。

情熱大陸

S.Sax. 吳政軒 / A.Sax. 黃翎茵
T.Sax. 余宛恩 / B.Sax. 許紘岳
Gt. 王浩宇 / Per. 李品樺

是一首充滿熱帶島嶼風情的電視節目主題曲，為小提琴家葉加瀬太郎所作的同名的日本旅遊節目「情熱大陸」之片頭曲，前奏帶有濃厚的探戈風格及全曲充滿熱情的拉丁律動感，整首歌傳達出勇敢、熱血而充滿鬥志的意象，帶領觀眾前往每一集新景點的冒險，此曲最困難的，首先是如何維持穩定的節奏，以及把切分音演奏得很精確，最後是聽起來很有味道，培養良好的技巧和律動感，是演奏這首曲子必須具備的基本技能。

- 中場休息 -

Rika Ishige （1982-）：A Miniature Town

石毛里佳：微型小鎮

B♭ Cl. 謝天艾 / 賴禹彤 / 石啟均 / 葉家瑜
B.Cl. 柳又誠

此曲的創作靈感是來自作曲家在平凡生活中看到的一幅畫。畫中描繪著一個縮小比例，色彩繽紛的小鎮景象，以此為起點，譜曲出從空中俯瞰小鎮的樣貌：街道蜿蜒、人聲鼎沸、孩子們奔跑玩耍，充滿童趣與活潑的日常映入耳中。

樂曲以明快的主題揭開序幕，各聲部間不斷交錯模仿，象徵小鎮的甦醒，街道逐漸熱鬧。作曲家將主導角色輪流給每位演出者扮演，展現人群互動使作品富有流動感。其中融合的斷奏、滑奏、強弱對比與節奏變化，呈現出鮮明的畫面感與合奏默契。

Miniature不只是尺寸上的縮小，也暗示著觀察世界的一種角度——從微觀中體會細膩的節奏與生命力，帶領聽眾進入一場精巧又活潑的微型世界之旅。

Satoshi Yagisawa（1975-）：Chignon

八木澤教司：髮髻

B♭ Cl. 賴禹彤 / 陳柏瑋 / 鄭緯
B.Cl. 柳又誠

Chignon一詞源自法語，意指包頭髮髻。作曲家八木澤教司在擔任合奏比賽評審時，注意到多位單簧管演奏者以髮髻造型登場，這一視覺印象激發了他創作此曲的靈感。他將這種優雅、柔美且富有層次的形象轉化為音樂語言，描繪如髮絲般細緻交織的線條，象徵女性的柔美、堅定與從容。此曲標題雖不具敘事性，卻暗藏極高的畫面感與情感深度，是極具旋律性與技巧性的作品。

樂曲開頭以柔美又溫暖的旋律鋪陳，低音聲部如髮絲般逐一疊入，營造層層繾綣之感。隨後進入節奏活潑、聲部交織的段落，旋律線輪流交替，而各部也都擁有短暫的主導角色。中間主題如互相梳理頭髮時，髮絲上的纏繞、疊合和流動，最後在輕盈而溫暖的氣氛中收尾，留下深刻的餘韻。接著樂曲開頭的主題再現，四人之間展現出流暢如絲線交織般的默契，再如收束髮絲般優雅而俐落的落幕。

Kazuhiko Morita（1952-2021）：Csárdás

森田一浩：查爾達斯舞曲

B♭ Cl. 石啟均 / 謝天艾 / 陳柏瑋
B.Cl. 柳又誠

查爾達斯舞曲（Csárdás）是義大利作曲家，維托里奧·蒙蒂（Vittorio Monti）於 1904 年創作的作品，其靈感來自匈牙利的傳統舞曲「Csárdás」，原意是「小酒館之舞」。原為小提琴與鋼琴的二重奏，後來被改編為各種樂器的版本。

樂曲以傳統「Csárdás」的雙段式結構為骨幹，慢板的 Lassan緩慢且抒情，用匈牙利民族音樂的抒情特質帶出自由而富情感的旋律；後段快板 Friska則是快速又熱情，以飛快的節奏、活潑的旋律與層層堆疊的推進，將氣氛推至高潮，展現舞曲的熱情與活力。

此次演出選擇的是森田一浩的單簧管四重奏版本，作曲家巧妙地將旋律重新分配至各聲部，不僅保留原曲的旋律魅力與節奏張力，還加入獨特的聲部分工與音色層次，將原曲的華麗旋律、強烈節奏與戲劇性起伏巧妙地分配至四個聲部，讓每位演奏者都能展現技巧與音樂性。

讓我們一起走進這段東歐風情的音樂旅程，感受查爾達斯舞曲從內斂到狂放的音樂魅力。

Yosuke Fukuda（1975-）：Pixels

福田洋介：像素

B♭ Cl. 柳又誠 / 賴禹彤 / 謝天艾
陳柏瑋 / 葉家瑜 / 鄭緯 / 石啟均

這首作品的創作靈感來自數位圖像最小的基本單位—像素。作曲家以音符作為聲音的元素，透過音型的排列組合、節奏的分解重構和音色的交織融合，呈現出如像素般的密集運動與幾何秩序，拼貼出一幅節奏鮮明、層次分明、色彩豐富的音樂畫面。

樂曲開頭由精準明確的短音音型反覆出現，藉由層層堆疊的斷奏，塑造像素般的點狀音響，接著由各聲部接力代入不同的旋律，時而流行時而搖擺，形成交錯呼應的動態感；中間抒情版的段落，對比前段重複性的機械式節奏，節奏趨緩且使用長音與和聲相互交織，利用音域間的轉換，造成空間感上不同的縱深層次；最後全體回到初段節奏動機，但音型更加緊湊，密集卻又條理分明，利用疊音、模仿與錯落般的聲響，製造「像素閃爍」般的音樂張力，熱烈壯闊的收尾。

TAKAHASHI,Hiroki（1979-）：The Three Emotions

高橋宏樹：三種情緒

E♭ Cl. 石啟均
B♭ Cl. 謝天艾 / 葉家瑜 / 鄭緯 / 賴禹彤
B.Cl. 陳柏瑋 / 柳又誠
Per. 李品樺

《The Three Emotions》分為三個樂章，作曲家利用單簧管七重奏的豐富音域與層次，詮釋出人類情感光譜中的三種截然不同的情緒狀態。

第一樂章《Swing》
呈現歡樂的情緒，樂章採快板（Allegro）速度，以節奏明快的主題展開，旋律線條靈動跳躍，富有青春與生命力。聲部之間頻繁的動機模仿與問答句式，營造出對話般的活潑氛圍。旋律多由高音部主導，中低音聲部則以切分節奏與碎型音群推動音樂前進。整體風格爽朗明快，如晨光乍現，喚醒萬物。

第二樂章《Waltz》
帶有一絲感傷與抒情，樂曲轉入抒情的慢板（Adagio），情感內斂細膩。旋律在各聲部間流轉，形成如歌般的線條與對位，彷彿獨白般的低語，帶出幽微而深沉的情緒層次。作曲家巧妙運用色彩和聲及長音音型，使整體音響更具厚度與延展性。此樂章極度考驗演奏者的音準掌控與聲音控制，亦是全曲情感張力的核心。

第三樂章《Latin》
則充滿熱情與活力，終樂章以急板（Allegro con fuoco）展開，節奏強烈、氣勢磅礴。作品在此邁向戲劇性高峰，透過緊湊的節奏動機與重複音型，堆疊出一種決然而堅定的音樂形象。聲部間穿插推進動機與呼應式旋律，節奏與速度不斷推升至結尾，終以全體總奏釋放壓抑已久的情緒能量，完成三種情緒的壯麗之旅。
